# 广东财经大学飞跃手册 — 产品需求文档 (PRD)

> **版本**: v1.1 | **状态**: 已发布 | **更新日期**: 2026-04-14

---

## 一、产品定位

| 维度 | 定义 |
|------|------|
| **产品名称** | 广东财经大学飞跃手册 |
| **产品类型** | 静态内容站点（Promotional Landing Page + 文章阅读系统） |
| **核心价值** | 消除信息差——由广财学子自发维护的真实经验分享平台 |
| **核心卖点** | ① 学生自发维护 ② 真实学长学姐经验 ③ PV 突破 10 万 ④ 关键节点决策辅助 |
| **目标用户** | 广财在读本科生（大一至大四），尤其是对转专业、保研/考研、出国留学等学业规划感到迷茫的学生；以及有意分享经验的学长学姐 |

---

## 二、User Stories

| ID | Story | AC（验收标准） | 状态 |
|----|-------|---------------|------|
| US-1 | 作为迷茫的大二学生，我想快速浏览不同类别的经验文章，以便找到与我当前困境相关的参考 | 首页有明确分类入口；分类页可按标签筛选；文章列表以列表形式展示标题+概要 | ✅ 已实现 |
| US-2 | 作为考研党，我想阅读某篇具体的考研经验文章，以便获取详细的时间线和备考策略 | 文章页排版清晰、中文阅读舒适；支持明暗主题切换；Markdown 渲染完整 | ✅ 已实现 |
| US-3 | 作为学长学姐，我想投稿自己的经验，以便帮助后来者 | 有明确的投稿引导入口（外链/二维码）；文章添加流程简单（新增 markdown 文件） | ✅ 已实现 |
| US-4 | 作为夜间浏览的用户，我想切换到暗色主题，以便保护眼睛 | 一键切换明暗主题；主题偏好持久化到 localStorage | ✅ 已实现 |
| US-5 | 作为移动端用户，我想在手机上流畅阅读 | 全站响应式；移动端导航折叠；文章页字号适配 | ✅ 已实现 |

## 三、信息架构

```
首页 (/)
├── Hero 区：品牌名 + Slogan + 核心数据（10万PV）
├── 特色区：4 个核心卖点卡片
├── 分类入口：转专业 / 保研 / 考研 / 出国留学 / 就业 / 其他
├── 最新文章：列表式展示最新 6 篇
└── CTA 区：投稿引导 + 社区共建

文章列表页 (/articles)
├── 分类筛选标签栏
├── 文章列表（标题+概要+元信息）
└── URL 参数支持 ?category=考研

文章详情页 (/articles/[slug])
├── 文章元信息（作者、日期、分类、标签）
├── Markdown 正文渲染
└── 上一篇/下一篇导航
```

## 四、设计系统

### 4.1 配色方案

**亮色主题（Light）**
| 角色 | 色值 | 用途 |
|------|------|------|
| 背景 | `#FAFAF5` | 米白色主背景 |
| 卡片/面板 | `#FFFFFF` | 卡片、面板浮层 |
| 强调色 | `#E5B044` | 黄色主强调（按钮、标签、高亮） |
| 强调色浅 | `#FFF8E1` | 黄色浅底（标签背景、hover） |
| 主文字 | `rgba(0,0,0,0.88)` | 标题、正文 |
| 次文字 | `rgba(0,0,0,0.55)` | 描述、辅助信息 |
| 弱文字 | `rgba(0,0,0,0.35)` | 时间戳、占位符 |
| 边框 | `rgba(0,0,0,0.08)` | 卡片边框、分割线 |
| 暖白交替 | `#F5F3EE` | 区块交替背景 |

| 强调色 | `#8B7CF6` | 蓝紫色主强调 |
| 强调色浅 | `rgba(139,124,246,0.12)` | 蓝紫色浅底 |
| 主文字 | `rgba(255,255,255,0.92)` | 标题、正文 |
| 次文字 | `rgba(255,255,255,0.60)` | 描述、辅助信息 |
| 弱文字 | `rgba(255,255,255,0.35)` | 时间戳、占位符 |
| 边框 | `rgba(255,255,255,0.08)` | 卡片边框、分割线 |
| 深色交替 | `#14122A` | 区块交替背景 |

### 4.2 字重层级

| 角色 | 字号 | 字重 | 行高 | 备注 |
|------|------|------|------|------|
| Hero 标题 | 48px | 700 | 1.15 | 短标题，一行优先 |
| 区块标题 | 28px | 600 | 1.25 | 编辑感，不张扬 |
| 列表项标题 | 17px | 600 | 1.35 | 列表式文章标题 |
| 正文 | 16px | 400 | 1.75 | 中文阅读舒适行高 |
| 辅助文字 | 14px | 400 | 1.60 | 描述、备注 |
| 标签/徽章 | 12px | 500 | 1.20 | 分类标签、元信息 |

### 4.3 组件样式

- **按钮**：圆角 8px，主按钮实心强调色，次按钮描边
- **文章列表项**：单行布局，左侧标题+概要，右侧元信息
- **标签**：圆角 999px（药丸形），浅底 + 强调色文字
- **留白节奏**：8px 基数，常用 8/16/24/32/48/64

## 五、技术方案

| 决策 | 选择 | 理由 |
|------|------|------|
| **框架** | Astro 6 | 零 JS 默认输出、内置 Markdown、文件路由、极致性能 |
| **样式** | CSS 变量 + scoped style | 无运行时、主题切换零成本、轻量 |
| **内容** | Astro Content Collections（glob loader） | 添加文章 = 新增 .md 文件，frontmatter 定义元信息 |
| **主题** | CSS 变量 + localStorage 持久化 | 纯 CSS 切换，无闪烁 |
| **部署** | 静态 HTML | 极致加载速度，可部署到任意平台 |
